# Senha Certa - Suas senhas num só lugar!

```
Este é um projeto criado a partir de um problema real que observo no dia a dia: a dificuldade de lidar com tantas senhas de bancos, cartões, aplicativos, streamings e outros serviços. Muitas vezes surge a dúvida sobre qual senha usar, onde ela foi anotada ou como está sendo armazenada.

Mesmo já existindo soluções, inclusive oferecidas por grandes empresas como o Google, a ideia de desenvolver este projeto surgiu como uma forma de propor uma alternativa, além de servir como um exercício prático para treinar, aprender e aprofundar meus conhecimentos em diferentes tecnologias, buscando sempre melhorar a forma como lidamos com esse problema.
